## 📖 About

This is a beginner-friendly command-line game developed in Python.

The objective is simple: guess the randomly generated number before running out of lives.

While simple, this project demonstrates fundamental programming concepts including loops, conditionals, exception handling, input validation, and random number generation.

## 🎮 Gameplay

The game generates a random number between **1** and **10**.

You start with **5 lives**.

After every incorrect guess, you'll receive a hint depending on how close you are to the correct number.

| Hint | Meaning |
|------|---------|
| 🔥 Burning | 1 number away |
| 🥵 Hot | 2 numbers away |
| 🙂 Warm | 3 numbers away |
| 🥶 Freezing | More than 3 numbers away |

Guess the number before your lives reach **0**.
